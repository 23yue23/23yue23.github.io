drain命令用于对某个node进行设定，是为了设定此node为维护做准备。
此命令主要执行的操作是：

  1. 设定此node不可以使用（cordon)
  2. evict（回收）了其上的两个pod

配置ingress

[root@node1 wordpress]# cat ingress.yaml
apiVersion: extensions/v1beta1
kind: Ingress
metadata:
name: wordpress-ingress
namespace: default
annotations:
kubernetes.io/ingress.class: “nginx”
spec:
rules:

  • host: wordpress.boshao.wang
    http:
    paths:
  • backend:
    serviceName: wordpress
    servicePort: 80

创建ingress

kubectl create -f ingress.yaml

查看ingres信息
